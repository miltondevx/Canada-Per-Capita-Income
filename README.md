# Income Analysis Project

This project analyzes canada-per-capita-income data using Python and Jupyter Notebook.

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization using matplotlib
- Basic statistical insights
- Predict Income for2020
## Tools Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib / LinearRegression
